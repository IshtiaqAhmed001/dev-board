Question-1: What are the different ways to select an element in the DOM?

There are five different ways to select an element. They are: document.getElementById("id"), document.getElementsByClassName("class"), document.getElementsByTagName("tag"), document.querySelector("selector"), document.querySelectorAll("selector")

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The difference between innerHTML, innerText and textContent are: innerHTML: Gets or sets HTML content inside an HTML element. innerText: Gets or sets only visible text inside an element. textContent: Gets or sets all text, including hidden content, inside an element.

Question-3: What is event delegation in the DOM?

Event delegation means attaching a single event listener to a parent element instead of adding event listeners to multiple child elements. When a child element is clicked, the event "bubbles up" to the parent, and the parent handles it.

Question-4: What is event bubbling in the DOM?

Event bubbling means that when an event happens on a child element, it automatically travels up (bubbles) to its parent in the DOM.
